#.github/workflows/build-push.yml



name: Docker Build & Push

on:
  push:
    branches: [ main ]
  pull_request:
    branches: [ main ]

jobs:
  docker:
    runs-on: ubuntu-latest

    steps:
      - name: Checkout source code
        uses: actions/checkout@v3

      - name: Set up Docker Buildx
        uses: docker/setup-buildx-action@v3

      - name: Login to Docker Hub
        uses: docker/login-action@v3
        with:
          username: ${{ secrets.DOCKER_USERNAME }}
          password: ${{ secrets.DOCKER_PASSWORD }}

      - name: Build and push Docker image
        uses: docker/build-push-action@v5
        with:
          context: .
          push: true
          tags: ${{ secrets.DOCKER_USERNAME }}/python-app:latest

9. Add GitHub secrets

In your GitHub repository:

Settings → Secrets and variables → Actions → New repository secret

Add these 2 secrets:

DOCKER_USERNAME = your_dockerhub_username
DOCKER_PASSWORD = your_dockerhub_password

Steps to get SECRET KEY (Access Token)
Step 1:

Go to:

https://hub.docker.com
Step 2:

Login with your account

Step 3:

Click your profile (top right)

Step 4:

Go to:

Account Settings → Security
Step 5:

Click:
👉 New Access Token

Step 6:

Fill details:

Token Name: github-action
Permission: Read & Write
Step 7:

Click Generate

Step 8:

👉 Copy the token (VERY IMPORTANT ⚠️)

This token = your SECRET KEY

🔑 Final Values

Use in GitHub:

DOCKER_USERNAME = your Docker Hub username
DOCKER_PASSWORD = your Access Token (secret key)
📍 Where to paste in GitHub

Go to GitHub:

Repository → Settings → Secrets and variables → Actions → New repository secret

Add both values.

10. How it works

When you push code to the main branch:

GitHub Actions starts automatically
It checks out your code
Sets up Docker Buildx
Logs in to Docker Hub
Builds Docker image
Pushes image to Docker Hub

Image name:

<your-dockerhub-username>/python-app:latest
11. Run the image locally after push
docker pull <your-dockerhub-username>/python-app:latest
docker run -d -p 5000:5000 --name mypythonapp <your-dockerhub-username>/python-app:latest

Open:

http://localhost:5000
