# ✅ 4. Install Docker

### Check:

```bash
docker --version
```

### If not installed:

```text
https://www.docker.com/products/docker-desktop/
```

After install:

```bash
docker run hello-world
```

✔ Output → confirms Docker working

---

# ✅ 5. Install Jenkins (via Docker – easiest)

(No need Java install)

```bash
docker volume create jenkins-data

docker run -d --name jenkins -p 8080:8080 -p 50000:50000 -v jenkins-data:/var/jenkins_home -v /var/run/docker.sock:/var/run/docker.sock jenkins/jenkins:lts
```

---

# ✅ 6. Install Docker inside Jenkins container

```bash
docker exec -u root jenkins bash -c "apt-get update && apt-get install -y docker.io"
```

---

# ✅ 7. Install kubectl

### Check:

```bash
kubectl version --client
```

### If not installed (Windows):

```bash
curl -LO "https://dl.k8s.io/release/v1.30.0/bin/windows/amd64/kubectl.exe"
```

# ✅ 8. Install Minikube (for Kubernetes)

### Check:


### Install:

```text
https://minikube.sigs.k8s.io/docs/start/
```

